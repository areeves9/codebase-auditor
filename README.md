# Codebase Auditor

Language-agnostic whole-codebase audit skill for security (OWASP 2025 / CWE Top 25), performance, DRY, and bug/exploit risk.

Phased workflow with hotspot prioritization for large repos. Produces `AUDIT_REPORT.md` and `REMEDIATION_PLAN.md`.

Compatible with: Claude Code, Gemini CLI, Vibe CLI, Qwen Code, OpenCode, Codex CLI.
